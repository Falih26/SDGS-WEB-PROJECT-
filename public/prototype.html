Dirancang untuk Koneksi Rendah

DesaNet dioptimalkan agar bekerja dengan koneksi seburuk 128kbps. Semua materi utama dapat diunduh saat ada koneksi dan digunakan tanpa internet sepenuhnya. Aplikasi menggunakan Progressive Web App (PWA) sehingga bisa diinstal dari browser tanpa Play Store.

Ukuran aplikasi kurang dari 5MB
Bekerja di ponsel Android lama (Android 6+)
Mode offline menyimpan hingga 2GB konten
Antarmuka tersedia dalam Bahasa Indonesia & 5 bahasa daerah
Sinkronisasi otomatis ketika koneksi tersedia

Teknologi yang Digunakan

DesaNet dibangun dengan teknologi yang terbukti handal, terjangkau untuk dioperasikan, dan mudah dirawat oleh teknisi lokal.

Frontend
Progressive Web App (PWA)
React Native (mobile)
IndexedDB untuk offline storage
Service Workers
Tailwind CSS
Backend & API
Node.js + Express
PostgreSQL Database
Redis (caching)
REST API + GraphQL
WebSocket (notifikasi)
Infrastruktur
MikroTik Router (WiFi desa)
Local edge server di desa
Cloud: AWS / GCP
CDN untuk konten statik
SMS gateway backup
